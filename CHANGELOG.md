# Changelog

## 1.2.0

- `/list` теперь доступен всем участникам группы, не только админам.
- Ежедневная памятка по умолчанию упоминает `/list` — посмотреть текущие варианты.

## 1.1.1

- Добавлена команда `/get_reminder` — посмотреть текущее время и текст ежедневной памятки.

## 1.1.0

- Ежедневная памятка про `/suggest`: команды `/set_reminder_time` и `/set_reminder_text`.
- Проверка на дубликаты в `/suggest` (по нормализованному тексту).
- Опрос теперь не анонимный, максимум 8 вариантов: до 7 реальных предложений + обязательный «Мимокрокодил».
- Исправлено: `GROUP_CHAT_ID` из настроек аддона теперь применяется всегда, а не только при первом запуске.
- Установка как аддон из GitHub-репозитория — обновления через кнопку Update.

## 1.0.0

- Первый релиз: `/suggest`, `/list`, `/edit`, `/delete`, `/create_poll`, `/set_schedule`, `/close_poll`, `/restore`, `/excluded`, `/start`.
