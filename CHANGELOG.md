# Changelog

## 1.5.0

- Памятка про `/suggest` теперь приходит не каждый день, а раз в 2 дня.

## 1.4.2

- `/help` теперь показывает разный список команд в зависимости от того, админ спрашивает или обычный участник.

## 1.4.1

- Добавлена команда `/get_schedule` — посмотреть текущее расписание автосоздания опроса.

## 1.4.0

- `/suggest` теперь уходит в очередь на модерацию (`pending`) вместо мгновенного добавления в список — админ одобряет/отклоняет кнопками прямо в DM-уведомлении.
- Отклонённые предложения хранятся как `rejected` и блокируют повторную отправку того же текста.
- `/help` и README обновлены под новый флоу.

## 1.3.2

- `/suggest` и `/edit` теперь отклоняют текст без единой буквы/цифры (например, только пробелы или только эмодзи).

## 1.3.1

- `/suggest` и `/edit` теперь отклоняют текст короче 3 символов и текст, начинающийся с `/` (например, случайно продублированную команду).

## 1.3.0

- «Мимокрокодил» больше не может выиграть опрос — его голоса не учитываются при определении победителя (только выражают «я не еду»).
- Добавлена команда `/help` со списком всех команд.

## 1.2.0

- `/list` теперь доступен всем участникам группы, не только админам.
- Ежедневная памятка по умолчанию упоминает `/list` — посмотреть текущие варианты.

## 1.1.1

- Добавлена команда `/get_reminder` — посмотреть текущее время и текст ежедневной памятки.

## 1.1.0

- Ежедневная памятка про `/suggest`: команды `/set_reminder_time` и `/set_reminder_text`.
- Проверка на дубликаты в `/suggest` (по нормализованному тексту).
- Опрос теперь не анонимный, максимум 8 вариантов: до 7 реальных предложений + обязательный «Мимокрокодил».
- Исправлено: `GROUP_CHAT_ID` из настроек аддона теперь применяется всегда, а не только при первом запуске.
- Установка как аддон из GitHub-репозитория — обновления через кнопку Update.

## 1.0.0

- Первый релиз: `/suggest`, `/list`, `/edit`, `/delete`, `/create_poll`, `/set_schedule`, `/close_poll`, `/restore`, `/excluded`, `/start`.
